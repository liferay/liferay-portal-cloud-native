# Checks for .ts or .tsx

Check | Category | Description
----- | -------- | -----------
TSConfigFileCheck | [Bug Prevention](bug_prevention_checks.md#bug-prevention-checks) | Performs several checks on `ts.config` file. |
TSSpecFileLocationCheck | [Bug Prevention](bug_prevention_checks.md#bug-prevention-checks) | Checks that `*.spec.ts` file should be inside a folder that contains a `config.ts`. |
TSSpecFileStylingCheck | [Styling](styling_checks.md#styling-checks) | Applies rules to enforce consistency in code style. |
WhitespaceCheck | [Styling](styling_checks.md#styling-checks) | Finds missing and unnecessary whitespace. |
