# Liferay Connectors

## Outbound

This connector can create or modify custom objects and other entities on Liferay. It reuses the base implementation of
[HTTP JSON Connector](https://github.com/camunda/connectors/blob/main/connectors/http/rest/README.md)
by providing a compatible element template.

## Inbound

This connector starts process instances on Liferay events. It is based on inbound webhooks.
